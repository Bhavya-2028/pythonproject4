# ECG Signal Generation using WGAN

## 📌 Description
This project generates synthetic ECG signals using a Wasserstein GAN (WGAN).

## 🚀 Features
- ECG signal generation using GAN
- Training loss visualization
- Saved trained model

## 🛠️ Tech Stack
- Python
- PyTorch
- NumPy
- Matplotlib

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run training:
   python src/train_wgan_ecg.py

## 📊 Output
- Training loss graph in `figures/`
- Generated ECG samples in `generated_ecg_samples/`

## 📁 Project Structure
- src/ → source code
- checkpoints/ → trained model
- figures/ → graphs
- generated_ecg_samples/ → outputs
